## 3. 상세 BM 설계

### 3.1 메타 성장 BM

#### 무료 구조
- 런 종료 시 연구 포인트 획득 (실패해도 지급)
- 일일 미션으로 추가 연구 포인트
- 기본 연구 트리 무료 해금

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 연구 포인트 팩 | 대량 연구 포인트 | $2.99~$9.99 |
| 연구 부스트 | 획득량 +50% (7일) | $1.99 |
| 연구 해금 패키지 | 특정 연구 즉시 해금 | $2.99~$4.99 |
| 실험 노트 확장 | 저장 슬롯 증가 | $1.99 |

### 3.2 마녀/캐릭터 BM

#### 무료 구조
- 기본 마녀 3종 무료
- 업적 달성으로 추가 마녀 2종 해금
- 이벤트에서 한정 마녀 획득 기회

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 마녀 해금 | 개별 마녀 구매 | $2.99~$4.99 |
| 마녀 번들 | 마녀 3종 + 스킨 | $9.99 |
| 마녀 스킨 | 외형 변경 (성능 동일) | $1.99~$3.99 |
| 시즌 마녀 | 한정 기간 판매 | $4.99 |

### 3.3 런 편의 BM

#### 무료 구조
- 일일 런 횟수 무제한
- 기본 스킬/유물 풀 접근 가능
- 실험 노트 기본 5슬롯

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 런 저장 슬롯 | 중간 저장 +2 | $1.99 |
| 재도전 토큰 | 같은 시드로 재도전 | 프리미엄 재화 50 |
| 스킬 재굴림 | 선택지 다시 뽑기 | 광고 or 재화 |
| 빌드 프리셋 | 선호 빌드 저장 +3 | $0.99 |

### 3.4 광고 BM

| 광고 유형 | 적용 | 보상 |
|----------|------|------|
| 리워드 광고 | O | 연구 포인트 2배, 스킬 재굴림, 부활 |
| 인터스티셜 | X | - |
| 광고 제거 패스 | O | $1.99 (영구) |

## 4. 시즌/이벤트 BM

### 시즌 패스
| 트랙 | 내용 | 가격 |
|------|------|------|
| 무료 트랙 | 연구 포인트, 기본 스킨 파츠 | 무료 |

## 6. BM 밸런스 원칙

### P2W 방지선
- 모든 콘텐츠 무과금 클리어 가능
- 마녀 간 성능 차이는 플레이 스타일 차이
- 메타 성장은 시간이 해결

### 과금 압박 최소화
- 런 횟수 제한 없음
- 실패해도 연구 포인트 지급
- RNG 스트레스는 광고로 완화
