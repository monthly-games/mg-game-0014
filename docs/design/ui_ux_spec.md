# MG-0014: Witch's Lab UI/UX 스펙

> **게임명**: Witch's Lab (마녀의 실험실)
> **장르**: 연금술 조합 RPG
> **카테고리**: Year 2 Core
> **문서 버전**: 1.0.0

---

## 1. 화면 설정

| 항목 | 값 |
|------|-----|
| **방향** | Portrait (세로) |
| **기준 해상도** | 1080 × 1920 |
| **지원 비율** | 16:9 ~ 21:9 |
| **타겟 FPS** | 60fps |

### 방향 선택 이유
- 조합 레시피 세로 배치
- 스킬 조합 UI 최적화
- 실험 결과 표시 용이

---

## 2. Safe Area

| 영역 | 값 | 용도 |
|------|-----|------|
| 상단 | 44-62dp | 자원, 마나 |
| 하단 | 34-48dp | 조합 슬롯, 실행 |
| 좌우 | 16dp | 콘텐츠 여백 |

### 3.1 조합 슬롯

| 요소 | 크기 | 설명 |
|------|------|------|
| 슬롯 | 72×72dp | 재료 배치 공간 |
| 조합 버튼 | 100×56dp | 조합 실행 |
| 결과 표시 | 80×80dp | 조합 결과 |

---

## 4. 터치 영역

| 동작 | 영역 | 제스처 |
|------|------|--------|
| 재료 선택 | 재료 목록 | 탭 |
| 재료 배치 | 재료 → 슬롯 | 드래그 |
| 조합 실행 | 조합 버튼 | 탭 |
| 레시피 확인 | 결과 영역 | 탭 |

### 터치 피드백
- 재료 배치: 슬롯 글로우
- 조합 시작: 가마솥 이펙트
- 조합 성공: 파티클 + 아이템 팝
