## 3. 1년차 라이브 플랜

### 3.1 초기 (1~3개월차)

| 주차 | 주요 활동 |
|------|----------|
| 1~2주 | 소프트 런칭, 버그 수정 |
| 3~6주 | 실험 노트 시스템 최적화 |
| 7~10주 | 기본 스킬/유물 풀 밸런스 |
| 11~12주 | 첫 실험 챌린지 준비 |

**핵심 과제:**
- 실험 노트 UX 최적화
- 빌드 다양성 확보
- 난이도 밸런스

### 3.2 중기 (4~8개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 4개월 | 주간 실험 챌린지 시작 |
| 5개월 | 월간 시즌 테마 (속성 중심) |
| 6개월 | 새로운 스킬/유물 세트 |
| 7개월 | 월간 시즌 테마 (스킬 중심) |
| 8개월 | 커뮤니티 빌드 추천 시스템 |

**핵심 과제:**
- 주간 챌린지 사이클 정착
- 시즌 테마 효과 검증
- 커뮤니티 기능 강화

### 3.3 후기 (9~12개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 9개월 | 빌드 공유/추천 시스템 확장 |
| 10개월 | 고난이도 연구 모드 |
| 11개월 | 시즌 테마 심화 |
| 12개월 | 1주년 실험 페스티벌 |

**핵심 과제:**
- 빌드 공유 시스템 완성
- 하드코어 콘텐츠 확대
- 커뮤니티 활성화
