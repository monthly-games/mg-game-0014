# game_0014 수익화 전략 - 마녀의 연구소: 실험형 퍼즐

## 게임 개요
- **장르**: 퍼즐 로그라이크 + 스킬 빌드/시너지
- **타깃**: 미드~하드코어, 전략/빌드 선호
- **수익화 우선순위**: IAP > 광고

---

## 1. 광고 전략

### 1.1 보상형 광고 배치

| Placement | 상황 | 보상 | 일일 제한 |
|-----------|------|------|----------|
| `run_continue` | 실험 실패 | 1회 부활 | 3회 |
| `reward_double` | 실험 종료 | 연구 포인트 2배 | 2회 |
| `skill_reroll` | 스킬 선택 | 리롤 1회 | 5회 |
| `daily_bonus` | 일일 보너스 | 2배 | 1회 |

### 1.2 전면 광고

- 최소화 (하드코어 친화)
- 실험 종료 후 5회 중 1회
- 구독자 제외
